# 📊 Motorola Mobile Sales Dashboard - Power BI

## 📌 Project Overview
This Power BI dashboard analyzes Motorola mobile sales data and provides business insights through interactive visualizations.

## 🛠 Tools Used
- Power BI
- Excel
- Power Query
- DAX

## 📈 Dashboard Features
- Total Sales
- Total Quantity
- Total Transactions
- Average Sales
- Brand Analysis
- City-wise Sales
- Payment Method Analysis
- Customer Ratings
